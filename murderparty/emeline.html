Emeline et la Belote

Résolver ces énigmes vous donnera un code à rentrer sur la page du dénouement

  Règles : 
      Suites logiques de carte à jouer
      
      Entrez un nombre à côté de chaque suite (un message validera la suite)
      Multiplier la valeur de la carte par la couleur : 
          Coeur = x2   Carreau = x3    Pique = x4    Trèfle = x5
      exemple : 3 de coeur, entrer la valeur 6 à côté de la suite
      
      Additionner tous les nombres de chaque suite vous donnera le code final pour Emeline
